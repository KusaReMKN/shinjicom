# ShinjiConf

## これはなに

PC と Seeeduino XIAO（MCU）越しに接続された E220-900T22S(JP)-EV2（通信モジュール）との間で単発の通信を行うプログラムです。

## 使い方

```
shinjiconf [-c] <LoRa>
```

*LoRa* には、MCU に対応する tty デバイスの名前を指定します。通信モジュールに送信するコマンドは、標準入力から読み込まれます。オプション **-c** を添えた場合、コマンドに続いてチェックサムを付与します。

## 実装上の注

標準入力から読み込まれるコマンドは 255 文字以内である必要があります。これ以上の入力が与えられた場合、それは黙って無視されます。

## 使用例

MCU が **/dev/ttyACM1** として見えている場合、次のように実行することで通信モジュールのデバイスアドレスを問い合わせられます。

```console
$ printf '\xC1\x00\x02' | ./shinjiconf -c /dev/ttyACM1 
to LoRa
0000:	c1 00 02 c3 
0x04 (4)
from LoRa
0000:	06 c1 00 02 01 00 c4 
0x07 (7)
```
